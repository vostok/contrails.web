## 🧰 Package Manager

We use **Yarn** as the package manager.

## 🟩 Node.js Version

Use **Node.js version 16.X**.  
For convenience, we recommend using [nvm (Node Version Manager)](https://github.com/nvm-sh/nvm).

## ▶️ Running the App

To start the application, run:

```bash
yarn install
yarn start
```

All other scripts can be found in the package.json file.
